# Pawsitive Haven - Foster Care Checklist

## Use This Checklist to Track Your Foster Journey

Print this document or use it as a reference to ensure you're completing all necessary tasks during your foster experience.

---

## Before You Pick Up Your Foster Dog

### Preparation Tasks

- [ ] **Review foster dog's information**
  - Name, age, breed
  - Medical history and current medications
  - Known behavioral notes
  - Special needs or requirements

- [ ] **Prepare your home**
  - [ ] Set up crate in quiet location
  - [ ] Prepare bedding (blankets, towels)
  - [ ] Identify potty spot outside
  - [ ] Remove hazards (toxic plants, small objects, chemicals)
  - [ ] Secure trash cans
  - [ ] Set up baby gates (if limiting access)

- [ ] **Gather supplies**
  - [ ] Confirm supplies from Pawsitive Haven (crate, leash, collar, food)
  - [ ] Purchase enzymatic cleaner
  - [ ] Get poop bags
  - [ ] Prepare treats for training
  - [ ] Optional: toys, puzzle feeders

- [ ] **Inform household members**
  - [ ] Review decompression rules with everyone
  - [ ] Assign responsibilities (who feeds, who walks)
  - [ ] Establish consistent rules (on furniture? which rooms?)

- [ ] **Plan for resident pets**
  - [ ] Separate spaces ready
  - [ ] Scent items for swapping
  - [ ] Baby gates for visual separation

- [ ] **Save emergency contacts in your phone**
  - [ ] Vet line: (555) PAW-VET1
  - [ ] Lost dog: (555) PAW-LOST
  - [ ] Foster support: (555) PAW-HELP

## At Pickup

### Collect All Items

- [ ] Foster dog on leash
- [ ] Pawsitive Haven collar with ID tag
- [ ] Crate
- [ ] Food (2-week supply)
- [ ] Food and water bowls
- [ ] Medical folder with:
  - [ ] Vaccination records
  - [ ] Spay/neuter status
  - [ ] Current medications
  - [ ] Upcoming vet appointments

### Ask Important Questions

- [ ] When did the dog last eat?
- [ ] When was the last potty break?
- [ ] Any known fears or triggers?
- [ ] Any recent medical treatments?
- [ ] How does the dog ride in cars?

### Verify Contact Information

- [ ] Confirm your regional coordinator's contact
- [ ] Confirm foster support email
- [ ] Confirm vet appointment process

---

## First 24 Hours

### Immediate Tasks

- [ ] Arrive home safely (direct route, no stops)
- [ ] Take directly to potty spot (on leash)
- [ ] Give 5-10 minutes to potty
- [ ] Bring inside to safe space/crate
- [ ] Offer water
- [ ] Offer small amount of food (may not eat - that's okay)
- [ ] Let dog rest - minimize interaction

### Environmental Setup

- [ ] Crate positioned in quiet area
- [ ] Water bowl accessible
- [ ] Potty schedule started (every 2-3 hours)

### Things to Avoid

- [ ] No introductions to other pets
- [ ] No visitors
- [ ] No off-leash time (even in fenced yard)
- [ ] No overwhelming with affection
- [ ] No loud noises or chaos

---

## First 72 Hours (Decompression Period)

### Daily Checklist

**Morning**
- [ ] Potty break (leashed)
- [ ] Fresh water
- [ ] Breakfast (note if eating)
- [ ] Brief calm interaction
- [ ] Crate time/rest

**Midday**
- [ ] Potty break (leashed)
- [ ] Check water
- [ ] Monitor behavior and mood
- [ ] Crate time/rest

**Evening**
- [ ] Potty break (leashed)
- [ ] Dinner (note if eating)
- [ ] Brief calm interaction
- [ ] Crate time

**Before Bed**
- [ ] Final potty break
- [ ] Settle in crate for night

### Track and Report

- [ ] Note eating/drinking amounts
- [ ] Note potty habits (accidents?)
- [ ] Note behavior (scared? curious? shut down?)
- [ ] Note any health concerns
- [ ] Report concerns to fostersupport@pawsitivehaven.org

---

## First Week

### Health Monitoring

- [ ] Eating normally (or improving)?
- [ ] Drinking water regularly?
- [ ] Normal stool (firm, not diarrhea)?
- [ ] No vomiting?
- [ ] Energy level appropriate?
- [ ] No coughing or sneezing?
- [ ] No limping or pain signs?

### Behavioral Observations

- [ ] Responding to name?
- [ ] Making eye contact?
- [ ] Showing curiosity about environment?
- [ ] Relaxing outside of crate?
- [ ] Accepting treats from you?

### Required Tasks

- [ ] Administer any medications as scheduled
- [ ] Keep medical records accessible
- [ ] Maintain consistent potty schedule
- [ ] Continue crate training

### Optional Tasks (If Dog is Ready)

- [ ] Begin name recognition training
- [ ] Start basic "sit" training
- [ ] Short, calm walks (on leash only)
- [ ] Very gradual introductions (resident pets - scent only)

---

## Week 2

### Resident Pet Introductions (If Applicable)

**Dogs:**
- [ ] Continue scent swapping
- [ ] Visual introduction through barrier
- [ ] First parallel walk (20+ feet apart)
- [ ] Closer parallel walks
- [ ] Brief controlled meeting (neutral territory)
- [ ] Always separate when unsupervised

**Cats:**
- [ ] Continue separation
- [ ] Visual through baby gate (foster dog on leash)
- [ ] Watch for prey drive - report any concerns

### Training Progress

- [ ] Name recognition improving?
- [ ] Sit on cue?
- [ ] House training improving?
- [ ] Leash manners improving?

### Marketing Tasks

- [ ] Take quality photos
- [ ] Submit photos to photos@pawsitivehaven.org
- [ ] Write personality description
- [ ] Submit bio to bios@pawsitivehaven.org

---

## Weeks 3-4

### Expanded Socialization (If Appropriate)

- [ ] Meet new people (one at a time)
- [ ] Visit new environments (start quiet)
- [ ] Practice car rides
- [ ] Handling exercises (touch ears, paws, mouth)

### Ongoing Training

- [ ] Continue sit, down, stay
- [ ] Work on leash walking
- [ ] Practice recall in secure area
- [ ] Crate remains positive

### Administrative

- [ ] Update foster team on progress
- [ ] Report any behavior changes
- [ ] Note what this dog needs in an adopter
- [ ] Update bio if personality has emerged more

---

## Monthly Tasks

### Medical

- [ ] Heartworm prevention given (note date: ________)
- [ ] Flea/tick prevention given (note date: ________)
- [ ] Any medication refills needed?
- [ ] Vet appointments scheduled/attended?

### Administrative

- [ ] Check in with foster support team
- [ ] Update availability status if needed
- [ ] Update photos if needed
- [ ] Report any adoption interest

### Supplies

- [ ] Food supply adequate? Request more if needed
- [ ] Any supplies needed? Email supplies@pawsitivehaven.org

---

## Semi-Annual Tasks (Long-Term Fosters)

### Health Check

- [ ] Schedule wellness check if no recent vet visit
- [ ] Ensure vaccinations are current
- [ ] Check heartworm test status
- [ ] Dental check needed?

### Documentation Update

- [ ] Update bio/description
- [ ] New photos
- [ ] Update any behavioral changes
- [ ] Review adoption requirements

---

## When Interest in Adoption Arises

### Preparation

- [ ] Review dog's adoption requirements with team
- [ ] Note any specific needs for adopter
- [ ] Be available for meet and greet scheduling
- [ ] Gather honest information to share with potential adopters

### Meet and Greet

- [ ] Schedule through meetandgreet@pawsitivehaven.org
- [ ] Choose appropriate location
- [ ] Prepare to answer questions honestly
- [ ] Have leash, treats, poop bags ready
- [ ] Let dog and adopter interact naturally

### Post Meet and Greet

- [ ] Provide feedback to adoption team
- [ ] Answer any follow-up questions
- [ ] Support additional meetings if needed

---

## Transition to Adoption

### Before Adoption Day

- [ ] Confirm adoption appointment scheduled
- [ ] Gather all dog's belongings
- [ ] Prepare medical records folder
- [ ] Write note for adopter (routine, quirks, tips)

### On Adoption Day

- [ ] Send dog clean and groomed
- [ ] Include comfort item with your scent (optional)
- [ ] Hand over all supplies and records
- [ ] Share final tips with adopter
- [ ] Say goodbye (it's okay to be emotional!)

### After Adoption

- [ ] Celebrate! You did it!
- [ ] Submit Happy Tails story to happytails@pawsitivehaven.org
- [ ] Rest and recover
- [ ] Update availability when ready for next foster
